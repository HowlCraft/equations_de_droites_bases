Définition d'une fonction affine

Definition : Définition

Une fonction affine est une fonction pouvant s'écrire sous la forme \(f(x)= mx+p\), avec \(m\) et \(p\) des nombres réels.

  • \(m\) est le coefficient directeur.

  • \(p\) est l'ordonnée à l'origine.

Example :

\(f : x \mapsto 3x - 5\) est une fonction affine, avec \(m=3\) et \(p=-5\)

\(g : x \mapsto 6+7x\) est une fonction affine, avec \(m=7\) et \(p=6\) (Attention : \(m\) est le nombre qui multiplie \(x\))

\(h : x \mapsto x^2 + 6\) n'est pas une fonction affine, car \(x\) est mis au carré.
